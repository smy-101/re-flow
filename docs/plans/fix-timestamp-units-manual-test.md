# fix-timestamp-units 手工验收用例

## 前置条件

- 本地已执行 `pnpm exec tsx lib/db/fix-timestamp-units.ts`
- `local.db` 中存在 AI 配置、工艺模板、管道数据
- 至少包含一组历史毫秒数据修复后的记录，以及一组通过当前 UI 或 API 新建的秒级记录

## AI 设置页

### 用例 1：历史配置时间显示正确
- **Given** 数据库中的 AI 配置记录曾经以毫秒值写入，且已被修复脚本归一为秒级
- **When** 用户访问 `/settings/ai`
- **Then** 配置卡片中的创建时间显示为真实当前年份
- **And** 不再出现 58149 年等异常未来日期

### 用例 2：错误时间显示正确
- **Given** 某个 AI 配置存在 `lastErrorAt`
- **When** 用户访问 `/settings/ai`
- **Then** 健康状态区域中的错误时间显示为真实本地日期
